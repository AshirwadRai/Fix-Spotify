# Settings

Every section, and what's in it.

## Playback

| Setting | What it does |
| --- | --- |
| **Autoplay** | When your queue runs low, tops it up with similar tracks. Off means the music stops at the end of the queue. |
| **Crossfade** | 0–12 seconds — one song melts into the next. |
| **Overlap** | Fine-tunes how the crossfade edges meet. |

## Sound

| Setting | What it does |
| --- | --- |
| **Equalizer** | Eight bands, eleven presets, custom curve — [full page](/guide/equalizer). |
| **Normalize volume** | Evens out loudness so a quiet track doesn't vanish after a loud one. |
| **Media quality** | Streaming bitrate: Auto · Normal · High · Very High. |

See **[Sound & Quality](/reference/sound)** for what these do to the audio.

## Sources

| Source | Default |
| --- | --- |
| **JioSaavn** | On |
| **SoundCloud** | On |
| **YouTube** | Off — opt-in, with a self-test on Android |

Turning YouTube on runs a real stream resolution on your device first. See [Finding Music](/guide/finding-music#turning-on-youtube).

## Storage

| Setting | What it does |
| --- | --- |
| **Download folder** | Where downloaded audio is written. |
| **Reset to default** | Puts it back if you've moved it somewhere the app can't write. |

## Updates

Checks for a new version. On Android the app installs it **in place**, which preserves your library — the behaviour you want, done for you.

See [Releases](/releases).

## Appearance

| Setting | What it does |
| --- | --- |
| **Quality / source badges** | Show or hide the small labels on each row. |
