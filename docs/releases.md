## How to update

### Android

Install the new APK **over** the old app — do not uninstall first.

**Settings → Updates** in the app will check for a new version and install it in place, which handles this correctly for you.

::: danger This one matters
Uninstalling the APK deletes your playlists, liked songs and history. Android wipes app data on uninstall and there's no way to get it back. Always install over the top.
:::

### Windows

Download and run the new installer. It replaces the previous version in place; your library is untouched.

**Settings → Updates** will tell you when there's a new one.

## Version numbering

Releases are tagged `vMAJOR.MINOR.PATCH`. The version shown in the app comes from the release tag itself, so the number you see in **Settings → About** always matches the release you installed.

## Reporting a problem with a release

Open an issue on **[GitHub Issues](https://github.com/AshirwadRai/Fix-Spotify/issues)** and include:

- The version from **Settings → About**
- Windows or Android, and the OS version
- What you did, and what happened instead
